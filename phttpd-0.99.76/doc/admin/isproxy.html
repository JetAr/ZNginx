Configuring "isproxy.conf"

This module establishes a proxy HTTP connection to another WWW server. It does not cache any data received.

Configuration options (isproxy.conf)

connect-error-page (string)
URL to a page to display in case an error occurs when trying to establish a connection.

proxy-table (table)
Table of proxy connections to establish, keyed on local URL's.

Example configuration file

	# isproxy.conf
	connect-error-page = /isproxy-error.html

	proxy-table = {
		/Lysator/	www.lysator.liu.se:80	/%{1-}
	}
