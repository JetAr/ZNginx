The PH module

This module can be used to query a PH/QI server for information.

Example request

	http://www.ifm.liu.se/ph/ns.liu.se?name=Peter+Eriksson

System issues

Causes the server to close Keep-Alive sockets after the data has been sent.

Configuration options (map.conf)

default-page (string)
URL to the default page to display if called without a query.

default-server (string)
Name or IP number of the host to connect if none specified in the query.

ph-labels (table)
List of labels to view in the result from the query instead of the returned labels.
