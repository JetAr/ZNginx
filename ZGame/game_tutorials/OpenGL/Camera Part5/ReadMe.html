Project Files

Source\Main.cpp (The Source File containing the main program code)
Source\Shader.cpp (This handles the management of shaders)
Source\Model.cpp (This handles the management of vertex data and rendering)
Source\GLFWManager.cpp (This handles the window and input code)
Source\TimerManager.cpp (This calculates the frames per second and time slice)
Source\Camera.cpp (This handles the camera movement for a 3D scene)
Source\InputManager.cpp (This handles the user input for things like the camera)
Headers\Main.h (The Header File containing the GLApplication definition)
Headers\Shader.h (This contains the Shader class definition)
Headers\Model.h (This the Model class definition)
Headers\WindowManager.h (This contains the WindowManager class definition)
Headers\TimeManager.h (This is the class definition for the TimeManager)
Headers\GLFWManager.h (This inherits the WindowManager for the GLFW library)
Headers\Camera.h (This is the Camera class definition)
Headers\InputManager.h (This is the InputManager class definition)
Shaders\Shader.vertex (This vertex shader file)
Shaders\Shader.fragment (This fragment shader file)
Camera5.vcxproj  (The Visual Studio Project File holding the project info)
Camera5.vcxproj.filters  (The Visual Studio Project File which stores the project folders)
Camera5.sln  (The Visual Studio Solution File holding the solution info)

Needed Libraries

opengl32.lib (The OpenGL Interface Library needed to compile the program)
glfw3.lib (A cross-platform library that displays a window and handles input)
glew (A cross-platform library that loads modern OpenGL functions and types)
glm (A cross-platform library that handles vector/matrix mathematics for us)

Instructions    

If you have Microsoft Visual Studio 2013 just click on the <Program Name>.sln file.  This will open up Visual C++.  You should then open the Source\Main.cpp for the start of the tutorial.  For Mac/Linux users, you will need to download the compiled library of GLFW for your OS, and then setup you own project that builds the source files depending on your IDE. If you are using Visual Studio, Hit Control-F5 to run the program.  If you have trouble, make sure you have the latest drivers for your video card in order to support OpenGL 4.  This page has those links:  https://www.opengl.org/wiki/Getting_Started
